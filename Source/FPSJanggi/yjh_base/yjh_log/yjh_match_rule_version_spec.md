# YJH 매치 버전 필드 명세서 (Draft)

## 0. 매치 버전 필드란?
- 전투 규칙 세트를 식별하는 버전 값.
- 동일 증상이라도 어떤 규칙 버전에서 발생했는지 분리 추적 가능하게 한다.

## 1. 필드 정의
- MatchRuleVersion: string (예: v0.1, v0.2)

## 2. 기록 위치
- CombatSession 시작 시 고정
- Finalize 이벤트/관측 지표/테스트 리포트에 함께 기록

## 3. 업데이트 규칙
- 규칙 변경(판정/종료 우선순위/타임아웃 정책)이 있으면 버전 증가
- 파괴적 변경은 major 증가(v1.x -> v2.0)
- 비파괴 추가는 minor 증가(v1.1 -> v1.2)

## 4. 운영 이점
- 회귀 분석 속도 향상
- 사용자 리포트와 내부 로그 매칭 용이
- A/B 룰 실험 기반 마련
